# Mine Sweeper
See it: http://cl-mine-sweeper.surge.sh/

## Technologies
* JavaScript
* react ^16.8.6
* react-dom ^16.8.6
* react-scripts 3.0.1
* styled-components ^4.2.0

## To run the app on local

Set up folder:
```
 git clone git@github.com:ivyc81/Mine_Sweeper.git
 cd Mine_Sweeper
 npm install
```

Start app:
```
 npm start # Open browser to localhost:3000 and try the app out!
```
